Dream AAPB



App Folders
-
- classes/
  Instantiatable classes that do a job, but don't appear on the view.
- components/
  React components, which do appear on the view.
- data/
  JSON datasets used by the application.
- routes/
  Remix route files for each of the application's webpage.
- utils/
  Non-instantiatable, portable utility classes.
- styles/
  CSS Styles used by the application.
